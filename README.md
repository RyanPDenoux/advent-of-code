# Introduction

Ongoing solutions for the [Advent of Code](https://adventofcode.com/2023/about) series of coding challenges. For now, I'm using it to deep-dive Golang, but I can see in the future expanding it to write solutions in other languages


<a id="org6b7e8f1"></a>

# Usage

Clone the repo and run/build the project. You will get a prompt to pick a day to which you have to give one of the days of the contest. For now this is hardcoded to pick from 2023 solutions.
